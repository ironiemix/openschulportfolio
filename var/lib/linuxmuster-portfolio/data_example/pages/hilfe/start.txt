====== Hilfe für Benutzer ======

===== Oberfläche =====

|**Logo**: Oben links befindet sich das Seitenlogo. Wenn man dieses anklickt gelangt man direkt zur Startseite des Wikis.| {{ :hilfe:logo.png |}} | 
|**Seitenname**: Oben rechts in doppelten eckigen Klammern wird der name der aktuellem Wiki-Seite angezeigt.|{{ :hilfe:actualpage.png |}}|
|** Aktueller Pfad:** Oben links unterhalb des Logos wird die Stelle angezeigt, an der Sie sich im Wiki derzeit befinden. Durch anklicken eines Begriffes gelangen Sie direkt dorthin. Begriffe, die rot dargestellt sind, zeigen auf Seiten, die (noch) nicht existieren.|{{ :hilfe:you_are_here.png |}}|
|**Hauptmenüknöpfe**:    |   |
|**Exportlinks**: Durch einen Klick auf diese Links kann die aktuelle Wikiseite in verschiedene andere Formate exportiert werden (HTML, PDF, Openoffice). Export nach Word gibt es nicht, jedoch gibt es Erweiterungen wür Word, mit denen man Openoffice Dokumente öffnen kann.| {{ :hilfe:exportbar.png |}}  |
|**Suchfeld**: Hier können Sie das Wiki nach Begriffen durchsuchen. Es werden alle Wikiseiten durchsucht, jedoch //nicht// die Inhalte verlinkter Dateien in anderen Formaten. | {{ :hilfe:searchfield.png |}} |
|**Schlagworte (Tags)**: Viele Seiten sind verschlagwortet, die Schlagworte, die der Seite zugeordnet sind werden oben rechts im Inhaltsbereich angezeigt. Durch Klick auf ein Schlagwort werden weitere Seiten mit demselbem Schlagwort angezeigt.| {{ :hilfe:tags.png |}} |
| **Seiteninhaltsverzeichnis**: Bei Seiten, die mehrere Überschriften haben, wird oben rechts im Inhaltsbereich automatisch ein Inhaltsverzeichnis angezeigt. Durch Klick auf eine Überschrift spring man direkt an die entsprechende Position auf der Seite. Sucrh Klick auf das "Minus"-Zeichen kann man das Menü ausblenden.  |{{ :hilfe:content.png |}} |
